# Project Notes

This repo publishes the CRFB Social Security taxation-of-benefits analysis as:

- a Next.js/Tailwind dashboard at `/`
- a Quarto manuscript at `/paper/`

The legacy Jupyter Book has been removed. Use `dashboard/` for the interactive
surface and `paper/` for the citable manuscript.

Useful commands:

```bash
make dashboard
make paper
make site
make test
```

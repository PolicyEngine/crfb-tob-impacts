# Reform Modeling Bible

The controlling CRFB reform-modeling plan lives at:

[`docs/current/REFORM_MODELING_BIBLE.md`](docs/current/REFORM_MODELING_BIBLE.md)

Before any paid Modal reform launch:

1. Read that file.
2. Read and update
   [`docs/current/reform-modeling-progress.json`](docs/current/reform-modeling-progress.json).
3. Do not launch unless the current gate permits it.

The production artifact is a **full reform output H5 in durable storage** for
each `(year, reform)` cell. Aggregate CSVs, selected cached arrays, and partial
Modal-volume artifacts are not production substitutes.
